Grafikkarten
1) Wenn Sie Intel-i810-Karten benutzen kann es sein, dass Sie mehr als den minimalen Speicher von 64MB benötigen.
2) Die ATI-Rage-Pro-8MB-Karte funktioniert eventuell im D3D-Modus mit den zur Zeit erhältlichen Treibern.
3) Manche Benutzer des Permedia-2-Chipsets können eventuell Probleme mit transperenz bekommen.
4) ALT+TAB funktioniert eventuell nicht richtig im Vollbildmodus.

Leistung
Benutzen Sie die Leistungeinstellung (unter Grafikoptionen), um die Leistung, zu kosten der Qualität der Grafik, zu erhöhen.

Bitte lesen Sie sich den Abschnitt über DirectX 7.0a in diesem Dokument genau durch. Er enthält wichtige Informationen über die Voraussetzungen für die erfolgreiche Ausführung des Spieles auf Ihrem Rechner.

* DIRECTX 7.0a *

Das Spiel verwendet DirectX 7.0a von Microsoft. Sie benötigen Treiber für Ihre Grafik- und Soundkarte, die die DX7a-Funktionen unterstützen, um das Spiel anwenden zu können. Bitte beachten Sie: Mit dem Spiel wird automatisch auch DirectX 7.0a installiert. Eine Aktualisierung Ihrer Grafik- und Soundkartentreiber auf DirectX 7a-Treiber erfolgt nicht. Sie müssen sich die neuesten DX7a-Treiber beim Hersteller Ihrer Karte selbst besorgen. 

In Teil II dieses Dokuments finden Sie eine ausführliche Beschreibung von DirectX.

* SPEICHER UND LEISTUNG *

Obwohl das Spiel auch mit 64 MB RAM gut läuft, verbessert sich die Leistung durch zusätzlichen Hauptspeicher deutlich.

Ungeachtet der tatsächlichen Menge an Arbeitsspeicher können Sie den verfügbaren Speicher maximieren und die allgemeine Leistung des Spiels steigern, indem Sie die folgenden Punkte beachten:
*  Schließen Sie alle anderen geöffneten Fenster.
*  Beenden Sie alle anderen Programme, einschließlich im Hintergrund ausgeführte Programme, wie ICQ.
*  Defragmentieren Sie Ihre Festplatte.

Sie haben darüber hinaus die Möglichkeit, durch bestimmte Einstellungen im Spiel die Framerate zu erhöhen. Benzutzen Sie dazu den Schieberegler für Leistungseinstellungen in dem Grafik-Menü: Hauptmenü / Konfiguration / Grafik.

* PRIMÄRE UND SEKUNDÄRE 3D-GRAFIKKARTEN *

Wenn auf Ihrem Rechner mehr als eine 3D-Grafikkarte installiert ist (z. B. eine primäre Karte und eine 3Dfx-Karte), können Sie nach dem Start des Spiels in dem Grafik-Menü die Grafikkarte (Rendering-Gerät) auswählen.

VORBEREITUNG IHRER FESTPLATTE

Um sicherzustellen, dass das Spiel problemlos funktioniert, sollten Sie sich vergewissern, dass Ihre Festplatte und Ihr Dateisystem optimal konfiguriert sind. Windows 95/98/ME/2000 wird mit zwei Hilfsprogrammen ausgeliefert, die Fehler suchen und beseitigen und die Leistung Ihrer Festplatte optimieren. 

Das erste dieser beiden Programme heißt ScanDisk. Es überprüft Ihre Festplatte auf Probleme und versucht, diese zu reparieren. Klicken Sie auf die START-Schaltfläche und wählen Sie im Menü Programme den Eintrag Zubehör. Klicken Sie hier auf Systemprogramme und schließlich auf ScanDisk, um ScanDisk zu starten.

Nachdem Sie ScanDisk ausgeführt haben, sollten Sie die Leistung Ihrer Festplatte optimieren, indem Sie diese defragmentieren. Klicken Sie auf die START-Schaltfläche in der Taskleiste von Windows 95/98/ME/2000 und wählen Sie im Menü Programme den Eintrag Zubehör. Nach einem Klick auf die Option Defragmentierung im Untermenü Systemprogramme wird die Defragmentierung gestartet.

INSTALLATION VON PROJECT IGI

Die Installation von Project IGI ist sehr einfach. Legen Sie die CD in das CD-ROM-Laufwerk ein. Bereits wenige Sekunden später erscheint der Startbildschirm.

Folgen Sie bitte den Anweisungen dieses Programms, um den Installationsprozess zu vervollständigen. Sollten Sie dazu aufgefordert werden, einen Neustart Ihres Rechners durchzuführen, tun Sie dies.

Sollte der Startbildschirm nach dem Einlegen der CD nicht erscheinen:

Doppelklicken Sie auf das Symbol ARBEITSPLATZ, anschließend auf das Symbol CD-ROM und schließlich auf die Datei SETUP.EXE, um den Startbildschirm aufzurufen ODER:

* Klicken Sie auf die Schaltfläche START.
* Wählen Sie im Startmenü Ausführen.
* Geben Sie an der Eingabeaufforderung d:\setup ein (d: entspricht hier Ihrem CD-ROM-Laufwerk).
* Klicken Sie auf OK, um mit der Installation des Programms zu beginnen.

Zunächst werden Sie aufgefordert, einen Pfad und einen Ordner anzugeben, in den Project IGI installiert werden soll. Wenn Sie möchten können Sie den Installationspfad selbstverständlich ändern.

Wenn DirectX 7.0a von Microsoft auf Ihrem Computer nicht installiert ist, erscheint der DirectX-Installationsbildschirm. Bitte lesen Sie die Bildschirmanweisungen genau durch, bevor Sie sich für eine Option entscheiden. Sie haben die Möglichkeit, DirectX nun zu installieren. Erkennt das Installationsprogramm von Project IGI eine ältere Version von DirectX, empfehlen wir Ihnen, DirectX nicht erneut zu installieren. Wenn das Installationsprogramm keine DirectX-Version erkennt, müssen Sie DirectX 7.0 installieren, damit Sie Project IGI spielen können. Weitere Informationen zu DirectX finden Sie im nächsten Abschnitt dieses Dokuments.

* CD-ROM PROBLEME *

Project IGI benötigt mindestens ein 4-fach-CD-ROM-Laufwerk mit 32-bit Treibern für Windows 95/98/ME/2000.

* Ich erhalte die Fehlermeldung "xxxxxx.xxx nicht gefunden", wenn ich Project IGI spiele oder installiere.
Diese Meldung erscheint normalerweise wenn Ihr Computer für das CD-ROM-Laufwerk anstatt der 32-Bit Windows 95-Treiber 16- Bit MS-DOS-Treiber verwendet. Sie können leicht prüfen, ob dies der Grund für die Fehlermeldung ist. Klicken Sie im Startmenü auf "Einstellungen" und anschließend auf "Systemsteuerung". In der SYSTEMSTEUERUNG wählen Sie "System" und im nun erscheinenden Fenster die Registerkarte Leistungsmerkmale. Hier erhalten Sie eine Zusammenfassung der Leistungsmerkmale Ihres Computers. Suchen Sie nach den Einträgen "Dateisystem: 32-bit" und "Das System ist optimal konfiguriert." 
Erscheint hier die Meldung "Laufwerk X benutzt Kompatibilitätsmodus für MS-DOS", bitten Sie den Verkäufer Ihres Systems um 32-Bit-Treiber für Ihr CD-ROM-Laufwerk.

* AUTOPLAY wird mehrmals gestartet
Bei einigen 24-fach-CD-ROM-Laufwerken wird die AutoPlay-Funktion von Project IGI mehrmals gestartet. Dies führt zu einer eingeschränkten Kontrolle während des Spiels. Um dieses Problem zu beheben, deaktivieren Sie auf der Registerkarte Einstellungen der Eigenschaften von CD-ROM-Laufwerk das Kontrollkästchen Automatische Benachrichtigung beim Wechsel.

* ABSTÜRZE *

* Wenn ich Project IGI, starte, verschwindet der Mauszeiger und mein Computer stürzt ab.
Vermutlich sind die Treiber für Ihre Soundkarte nicht DirectX-kompatibel. Wenden Sie sich in diesem Fall an den Hersteller Ihrer Soundkarte, um DirectX-7-Teiber zu erhalten.

* Sobald ich Project IGI starte, erhalte ich die Fehlermeldung:
  "Die Anwendung Project IGI.exe verweist auf den Adressraum xxxx:xxxx. Dieser kann nicht gelesen werden."
Vermutlich sind die Treiber für Ihre Grafikkarte nicht DirectX-kompatibel. Wenden Sie sich in diesem Fall an den Hersteller Ihrer Grafikkarte, um DirectX-7a-Teiber zu erhalten.

* Die Installation bricht immer an derselben Stelle ab. Ich kann daher Project IGI nicht installieren.

Es gibt drei mögliche Gründe für dieses Problem:
1) Sie haben auf Ihrer Festplatte nicht genügend freien Speicherplatz. Bitte entfernen Sie nicht benötigte Programme, um zusätzlichen Speicherplatz zu erhalten. Installieren Sie Project IGI anschließend erneut.
2) Einige Dateien wurden möglicherweise in fehlerhafte Bereiche Ihrer Festplatte kopiert. Sollte dies der Fall sein, starten Sie ScanDisk und führen Sie eine intensive Überprüfung durch. Wenn Sie die Meldung erhalten, dass alle Fehler auf Ihrer Festplatte behoben sind, installieren Sie das Spiel erneut.
3) Möglicherweise befinden sich Fingerabdrücke oder Schmutz auf der CD. Untersuchen Sie die Unterseite der CD. Sollten Sie dort Verunreinigungen finden, reinigen Sie die Oberfläche mit einem sauberen, weichen Tuch, indem Sie von der Mitte nach außen wischen.

* Project IGI stürzt ohne Fehlermeldung ab.

Dieses Problem kann verschiedene Ursachen haben. Im Folgenden finden Sie eine Liste möglicher Lösungen:

1) Vergewissern Sie sich, dass die CD-ROM sauber ist (suchen Sie nach Kratzern und Verunreinigungen auf der Oberfläche der CD).
2) Vergewissern Sie sich, dass das Spiel einwandfrei installiert wurde. 
3) Stellen Sie sicher, dass DirectX 7.0a auf Ihrem System installiert ist und korrekt arbeitet.
4) Stellen Sie sicher, dass Sie die aktuellsten Windows-95/98-Treiber für Ihre Grafikkarte besitzen. Diese müssen DirectX 7.0a unterstützen.
5) Prüfen Sie, ob Sie die aktuellsten Windows-95/98-Treiber für Ihre Soundkarte verwenden. Diese müssen DirectX 7.0a unterstützen.
6) Aktivieren Sie den virtuellen Speicher. 
7) Führen Sie Scandisk aus. 
8) Defragmentieren Sie Ihre Festplatte. 
9) Löschen Sie alte temporäre (.TMP) Dateien im Verzeichnis C:\WINDOWS\TEMP oder C:\WIN95\TEMP (über den Windows Explorer) von der Festplatte.
10) Stellen Sie sicher, dass keine Anti-Viren-Programme (wie Norton's AntiVirus) im Hintergrund ausgeführt werden, während Sie Project IGI spielen.
11) Benutzen Sie keine Programme zur Speicherverwaltung von Drittanbietern (wie QEMM 8.0 für Windows 95 von QuarterDeck), während Sie Project IGI spielen. 
12) Beenden Sie alle Disk Cache-Programme von Drittanbietern, bevor Sie Project IGI starten.
13) Deaktivieren Sie den Nortons Crash Protector, bevor Sie Project IGI ausführen. 
14) Deinstallieren Sie das Spiel und installieren Sie es erneut.
15) Beenden Sie das Spiel, booten Sie Ihren Rechner neu und starten Sie das Spiel dann erneut.

Sie sollten Ihren Computer nach einem Absturz von Project IGI neu starten, bevor Sie das Spiel erneut ausführen, da nach einem Absturz DirectDraw und DirectSound möglicherweise nicht verfügbar sind. Dadurch kann das Spiel nicht auf Ihre Sound- und Grafikkarte zugreifen.

* Fragen zur AutoPlay-Funktion *

Warum wird die AutoPlay-Funktion nicht aktiviert, wenn ich die CD Project IGI in das CD-ROM-Laufwerk einlege?

Hier handelt es sich meist um ein Konfigurationsproblem. Die AutoPlay-Funktion von Windows 95/98/ME/2000 lässt sich auf unterschiedliche Weise aktivieren. Standardmäßig gehen Sie folgendermaßen vor:

1) Öffnen Sie die SYSTEMSTEUERUNG. Klicken Sie dazu auf die START-Schaltfläche in der Taskleiste und rufen Sie im Startmenü den Eintrag Einstellungen und anschließend Systemsteuerung auf.
2) Doppelklicken Sie auf den Eintrag SYSTEM, um die Registerkarte "Eigenschaften von System" aufzurufen.
3) Klicken Sie auf die Registerkarte Geräte-Manager. Suchen Sie in der Liste den Eintrag CD-ROM und klicken Sie auf das Pluszeichen ("+"). (Erscheint hier ein Minuszeichen, klicken Sie dieses bitte nicht an.) 
4) Doppelklicken Sie auf das CD-ROM-Laufwerk, um das Fenster Eigenschaften von CD-ROM-Laufwerk aufzurufen.
5) Wählen Sie die Registerkarte Einstellungen. 
6) In der Mitte der Registerkarte befinden sich im Abschnitt Optionen einige Kontrollkästchen. Im unteren Teil dieses Abschnitts finden Sie ein Kontrollkästchen mit dem Titel Automatische Benachrichtigung beim Wechsel.
7) Aktivieren Sie dieses Kontrollkästchen und klicken Sie auf OK, um den Vorgang abzuschließen.

* Was ist DirectX und muss ich es installieren, damit ich Project IGI spielen kann?

DirectX ist ein Produkt von Microsoft, das die optimale Kommunikation zwischen Windows 95/98/ME/2000 sowie Soft- und Hardwarekomponenten ermöglicht. Hersteller von Grafik- und Soundkarten entwickeln deshalb spezielle Treiber, die DirectX unterstützen. 

Project IGI unterstützt ferner DirectDraw, DirectInput und Direct3D, welche Komponenten von DirectX sind. Wenn Sie DirectDraw- oder Direct3D-Treiber auf Ihrem Rechner installiert haben, oder Sie die DirectX 7.0a-Treiber, die auf der Spiel-CD enthalten sind, installiert haben und dennoch Probleme auftauchen, sollten Sie sich an den Hersteller Ihrer Grafikkarte wenden, und die aktuellen Treiber installieren.
Vor allem die Treiber für Grafikkarten werden etwa alle 2-3 Monate aktualisiert. Je nach Grafikkarte und Hersteller haben Sie also gute Chancen, häufig aktuelle Treiber zu erhalten. Wenn Sie noch keinen Internetzugang haben, empfehlen wir Ihnen dringend, sich einen Zugang zu verschaffen. Aktuelle Treiber können nämlich in der Regel problemlos von den Homepages der verschiedenen Anbieter heruntergeladen werden.
Project IGI unterstützt ferner DirectSound sowie zahlreiche andere neue Soundfunktionen von DirectX 7.0a. Wenden Sie sich an den Hersteller Ihrer Soundkarte, um aktuelle, DirectX 7.0a-kompatible Windows 95/98/ME/2000-Treiber zu erhalten.
	
* Installation von DirectX
DirectX 7.0a von Microsoft ist im Lieferumfang von Project IGI enthalten. Sie haben die Möglichkeit, DirectX gleichzeitig mit dem Spiel zu installieren. Klicken Sie dazu auf die entsprechende Schaltfläche des Installationsprogramms. Sie können die Installation von DirectX jederzeit nachholen.

* Woran erkenne ich, welche Version von DirectX auf meinem System installiert ist? Läuft das Spiel auch mit einer älteren Version? Falls ich eine ältere Version von DirectX habe, wo bekommen ich die neueste Version?
Das Installationsprogramm von Project IGI prüft, automatisch, welche DirectX-Version auf Ihrem System installiert ist. DirectX 7a ist die aktuellste Version, die auch mit Project IGI ausgeliefert wird und ausgiebig getestet wurde. Für die korrekte Ausführung von Project IGI ist DirectX 7a ERFORDERLICH. Wenn Sie eine ältere Version von DirectX installiert haben, lässt sich das Spiel nicht starten.
	
* Wie kann ich die DirectX-Treiber manuell installieren?
Wenn Sie Microsofts DirectX 7.0a MANUELL installieren möchten, befolgen Sie diese Anweisungen: 

1) Öffnen Sie den Windows-Explorer (klicken Sie auf die START-Schaltfläche und wählen Sie PROGRAMME, WINDOWS-EXPLORER).  
2) Legen Sie die Project IGI-CD in das CD-ROM-LAUFWERK (wenn die AUTORUN-Funktion aktiviert wird, wählen Sie die Schaltfläche CD ansehen, um zum Explorer zurückzukehren. Überspringen Sie Punkt 3). 
3) Öffnen Sie das CD-ROM-Laufwerk (normalerweise D:\) und öffnen Sie den Ordner DXSetup. 
4) Doppelklicken Sie auf die Datei DXSETUP.EXE, um die Installation von DirectX manuell zu starten. 
5) Nach einigen Sekunden erscheint nun möglicherweise ein Fenster namens DirectX Setup.
6) Klicken Sie hier auf die Schaltfläche DirectX erneut installieren, um die Installation zu starten. 
Eventuell erscheint folgende Meldung:
"Setup hat entdeckt, dass Ihre Anzeigetreiber mit DirectX nicht getestet wurden. Um eine optimale Leistung des Spiels zu gewährleisten, kann Setup die vorhandenen Treiber ersetzen. Möchten Sie die vorhandenen Treiber ersetzen?"  
Wählen Sie hier NEIN, damit Sie Ihre ursprünglichen Grafiktreiber nicht überschreiben oder beschädigen. Hinweis: Wiederholen Sie den Vorgang und wählen Sie die Option JA, wenn das Spiel nach der Installation der Treiber immer noch nicht korrekt ausgeführt wird. Denken Sie daran: Wenn Probleme auftreten, können Sie die Originaltreiber jederzeit wiederherstellen.
7) Sie werden nun aufgefordert, den Computer neu zu starten. Wählen Sie an der Eingabeaufforderung JA, um die neuen Treiber zu initialisieren. 

* Hilfe! Project IGI hat mein System durcheinander gebracht und ich glaube, dass DirectX daran Schuld ist. Wie kann ich meine ursprünglichen Treiber wiederherstellen? 
DirectX ist ein Standard für Windows 95/98/ME/2000-Anwendungen. Da nahezu jede Hochleistungssoftware diese Technologie nutzt, raten wir normalerweise davon ab, die DirectX-Komponenten eines Systems zu verändern. Leider treten auf einigen Systemen Probleme mit DirectX auf, die wiederum Treiberprobleme nach sich ziehen. SIE KÖNNEN DIRECTX NICHT VON IHREM SYSTEM ENTFERNEN, aber Sie haben die Möglichkeit, die ursprünglichen Audio- und Grafiktreiber, die bei der Installation von DirectX ersetzt wurden, wiederherzustellen.

Wenn DirectX 7.0a auf Ihrem Computer installiert ist, stellen Sie die Originaltreiber in der Regel über die Registerkarte INSTALLIEREN/DEINSTALLIEREN in der Windows-SYSTEMSTEUERUNG wieder her. Weitere Informationen dazu finden Sie im Abschnitt DirectX-Version.

Nachdem Sie in der SYSTEMSTEUERUNG das Symbol SOFTWARE ausgewählt haben, erscheint auf der Registerkarte Installieren/Deinstallieren eine Liste aller unter Windows 95/98/ME/2000 registrierten Programme.

Doppelklicken Sie nun auf den Eintrag "DirectX-Treiber", um das Fenster DirectX Setup aufzurufen. Klicken Sie anschließend auf die Schaltfläche Anzeigentreiber wiederherstellen, um die ursprünglichen Grafiktreiber wiederherzustellen. Klicken Sie anschließend auf Audiotreiber wiederherstellen, wenn Sie auch die ursprünglichen Audiotreiber wieder einrichten möchten. Dadurch werden die älteren, bei der Installation von DirectX entfernten, Treiber wiederhergestellt. DirectX wird dabei NICHT entfernt. Wenn Sie das Programm nun immer noch nicht starten können, wenden Sie sich bitte an den Hersteller Ihres Systems, um DirectX-kompatible Grafik- und Soundtreiber zu erhalten ODER besuchen Sie die Webseite des entsprechenden Herstellers.

* Funktioniert das Spiel auch ohne DirectX?
Ich kann auf meinem Computer DirectX nicht verwenden! Gibt es eine andere Möglichkeit, Project IGI auszuführen?  
Falls DirectX auf Ihrem System nicht installiert ist, kann Project IGI leider NICHT ausgeführt werden.
